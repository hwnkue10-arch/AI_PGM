# Python Practice 정리

이 문서는 파이썬 기초 문법과 실습 내용을 정리한 것이다.  
조건문, 반복문, 함수, 예외 처리, 파일 입출력, 가변 인자 등을 예제로 익히는 것을 목표로 한다.

---

## 1. 조건문

조건문은 특정 조건에 따라 다른 코드를 실행할 때 사용한다.  
파이썬에서는 `if`, `elif`, `else`를 사용한다.

### if / elif / else
- `if`: 조건이 참일 때 실행한다.
- `elif`: 앞 조건이 거짓이고, 추가 조건을 검사할 때 사용한다.
- `else`: 위 조건이 모두 거짓일 때 실행한다.

예를 들어 성적이 `A`, `B`, `C`, `D`인지에 따라 다른 메시지를 출력하는 코드를 작성할 수 있다.

### match / case
파이썬 3.10 이상에서는 `match` / `case` 문을 사용할 수 있다.  
이는 여러 경우를 분기할 때 `if`보다 더 읽기 쉽게 표현할 수 있다.

예를 들어:
- `grade` 값이 `A`이면 A 관련 메시지 출력
- `grade` 값이 `B`이면 B 관련 메시지 출력
- 그 외의 값은 기본 처리

또한 숫자를 기준으로 분기하는 예제도 포함되어 있으며,  
점수를 한 자리 수로 변환해 등급을 판별하는 방식도 실습했다.

---

## 2. 반복문

반복문은 같은 작업을 여러 번 수행할 때 사용한다.  
파이썬에서는 `while`과 `for`를 주로 사용한다.

### while문
`while`문은 조건이 참인 동안 계속 반복한다.

실습 내용:
- 1부터 10까지 출력하기.
- 특정 조건에서 반복을 멈추는 `break` 사용.
- 특정 조건에서 현재 반복만 건너뛰는 `continue` 사용.
- `while ... else` 구조를 통해 반복이 정상 종료되었는지 확인.

### 중첩 반복문
반복문 안에 또 다른 반복문을 넣어 사용할 수 있다.  
이 구조는 구구단, 좌표 출력, 도형 출력 등에 자주 사용된다.

실습 내용:
- 구구단 출력
- 좌표 형태의 값 출력
- 삼각형 또는 계단형 모양 출력

### for문
`for`문은 반복 가능한 객체를 순회할 때 사용한다.

실습 내용:
- `range()`를 사용한 숫자 반복
- 1부터 10까지 출력하기
- 합계 계산하기
- 짝수만 골라 출력하기

예를 들어 1부터 100까지의 짝수 합을 구하는 방식도 실습했다.

## 3. 함수

함수는 자주 쓰는 코드를 묶어서 재사용하기 쉽게 만든 것이다.  
입력을 받아 결과를 반환할 수도 있고, 출력만 수행할 수도 있다.

### 기본 함수
`def` 키워드로 함수를 정의한다.  
예를 들어 두 수를 더하는 함수, 곱하는 함수, 나누는 함수를 만들 수 있다.

### 매개변수와 반환값
- 매개변수: 함수가 받는 값
- 반환값: 함수가 돌려주는 결과

나눗셈 함수에서는 0으로 나누는 경우를 검사하여 오류를 직접 발생시키는 방식도 포함되어 있다.

### 여러 개의 인자
`*args`를 사용하면 여러 개의 위치 인자를 받을 수 있다.  
예제에서는 전달된 숫자들을 모두 더하는 함수가 있었다.

### 키워드 인자
`**kwargs`를 사용하면 이름이 있는 인자를 딕셔너리 형태로 받을 수 있다.  
예제에서는 `name`, `age`, `city` 같은 값을 받아 출력했다.

### lambda 함수
`lambda`는 짧은 함수를 한 줄로 작성할 때 사용한다.  
간단한 덧셈 같은 기능에 적합하다.

### 변수 범위
- 지역 변수: 함수 안에서만 사용되는 변수
- 전역 변수: 함수 밖에서도 접근 가능한 변수

실습에서는 `global` 키워드를 사용해 전역 변수를 수정하는 예제도 다뤘다.

## 5. 예외 처리

예외 처리는 실행 중 발생할 수 있는 오류를 안전하게 다루는 방법이다.  
파이썬에서는 `try`, `except`, `else`, `finally`를 사용한다.

### try / except
오류가 날 수 있는 코드를 `try`에 넣고,  
실패했을 때는 `except`에서 처리한다.

### else / finally
- `else`: 예외가 없을 때 실행
- `finally`: 예외 발생 여부와 관계없이 항상 실행

실습에서는 계산기 예제를 통해 다음을 다뤘다.
- 덧셈
- 뺄셈
- 곱셈
- 나눗셈
- 0으로 나누는 경우의 예외 처리

## 6. 자료형과 순회

반복문에서는 여러 자료형을 순회할 수 있다.

### iterable
순회 가능한 객체로는 다음이 있다.
- `list`
- `tuple`
- `dict`
- `set`
- `str`

### 딕셔너리
딕셔너리는 `key`와 `value`로 구성된다.  
실습에서는 딕셔너리의 키만 순회하거나, `items()`로 키와 값을 함께 출력했다.

### tuple
튜플은 여러 값을 묶는 자료형이다.  
함수에서 여러 값을 반환할 때도 사용된다.

## 7. 파일 입출력

파일을 다루는 기능은 데이터를 저장하고 읽을 때 중요하다.

### 파일 열기
`open()`을 사용해 파일을 연다.  
모드에는 다음이 있다.
- `r`: 읽기
- `w`: 쓰기
- `a`: 이어쓰기

### 파일 쓰기
파일에 문자열을 저장하는 실습을 했다.  
반복문으로 여러 줄을 작성하는 방법도 포함되어 있다.

### 파일 읽기
- `readline()`: 한 줄 읽기
- `readlines()`: 여러 줄을 리스트로 읽기

### 경로와 디렉터리
`os.makedirs()`를 사용해 폴더를 만들 수 있다.  
경로를 문자열로 직접 입력할 때는 `r""` raw string을 사용하면 백슬래시 문제를 줄일 수 있다.

## 8. 명령줄 인자

`sys.argv`를 사용하면 실행할 때 외부에서 값을 전달받을 수 있다.

실습에서는:
- `sys.argv[0]`: 실행 파일 이름
- `sys.argv[1:]`: 사용자가 전달한 인자들

을 확인했다.
