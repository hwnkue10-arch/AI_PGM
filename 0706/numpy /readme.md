# 📘 NumPy 학습 정리

NumPy는 Python에서 **수치 계산과 배열 연산을 빠르게 처리하기 위한 핵심 라이브러리**이다.

---

# 1. NumPy란?

NumPy(Numerical Python)는 다차원 배열(ndarray)을 기반으로 한 수치 계산 라이브러리이다.

## 특징

- 빠른 배열 연산
- 벡터화(Vectorization) 지원
- 다차원 배열 지원 (1D, 2D, 3D ...)
- 과학 계산 및 데이터 분석에 필수

---

# 2. NumPy 배열 (ndarray)

NumPy의 핵심 자료구조는 ndarray이다.

## 리스트 vs NumPy 배열

| Python List | NumPy Array |
|-------------|------------|
| 느림 | 빠름 |
| 반복문 필요 | 벡터 연산 가능 |
| 일반 자료형 | 수치 연산 최적화 |

---

# 3. NumPy 배열 생성

## 기본 생성

- 리스트를 NumPy 배열로 변환하여 생성

## 특징

- 같은 자료형으로 자동 통일
- 메모리 효율적

---

# 4. 배열의 속성

NumPy 배열은 다음과 같은 정보를 가진다.

## 주요 속성

- shape: 배열의 구조 (행, 열)
- ndim: 차원 수
- size: 전체 요소 개수
- dtype: 데이터 타입

---

# 5. 인덱싱과 슬라이싱

NumPy 배열은 리스트와 유사하게 인덱싱이 가능하다.

## 특징

- 1차원, 2차원 모두 지원
- 행/열 단위 접근 가능

---

# 6. 브로드캐스팅 (Broadcasting)

서로 다른 크기의 배열 간에도 연산이 가능하도록 자동 확장되는 기능이다.

## 특징

- 작은 배열이 자동으로 확장됨
- 반복문 없이 연산 가능

---

# 7. 벡터화 연산 (Vectorization)

NumPy의 가장 중요한 개념이다.

## 개념

반복문 없이 배열 전체에 연산을 한 번에 수행하는 방식이다.

---

## 예시 개념

### 일반 Python 방식

- 반복문을 사용하여 요소별 계산 수행

### NumPy 방식

- 배열 전체를 한 번에 연산

---

## 장점

- 속도가 매우 빠름
- 코드가 간결함
- 가독성이 높음

---

## 벡터화 연산 종류

### 1) 기본 사칙연산

- 배열 + 숫자
- 배열 + 배열
- 배열 * 스칼라

---

### 2) 배열 간 연산

- 같은 위치끼리 계산
- element-wise 연산

---

### 3) 조건 연산

- 조건을 만족하는 값만 선택
- Boolean indexing 활용

---

# 8. 유용한 NumPy 함수

## 1) zeros

- 0으로 채워진 배열 생성

## 2) ones

- 1로 채워진 배열 생성

## 3) arange

- 일정 범위 숫자 생성

## 4) linspace

- 일정 간격으로 숫자 생성

---

# 9. 배열 변형

## reshape

- 배열의 형태를 변경

## 특징

- 데이터는 그대로 유지
- shape만 변경

---

# 10. 통계 함수

NumPy는 기본 통계 기능을 제공한다.

## 주요 함수

- mean (평균)
- sum (합)
- min / max
- std (표준편차)

---

# 11. 2차원 배열 (Matrix)

NumPy는 행렬 연산을 지원한다.

## 특징

- 행렬 구조 지원
- 수학적 계산 가능
- 머신러닝 핵심 구조

---

# 12. 행렬 연산

## 기본 연산

- 덧셈
- 뺄셈
- 곱셈 (element-wise)

## 행렬 곱셈

- dot product 사용

---

# 13. 난수 생성

NumPy는 랜덤 데이터 생성 기능을 제공한다.

## 주요 기능

- random.rand
- random.randint
- random.randn

---

# 📌 핵심 요약

- NumPy는 고속 수치 계산 라이브러리
- ndarray는 NumPy의 핵심 자료구조
- 벡터화 연산이 가장 중요한 개념
- 반복문 없이 배열 전체 연산 가능
- 브로드캐스팅으로 자동 크기 확장
- 행렬 연산과 통계 기능 제공

---
