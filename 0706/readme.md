# Python 객체지향 개념 정리

## 1. 클래스와 객체

-   클래스(Class): 객체를 만들기 위한 설계도
-   객체(Object): 클래스로 생성한 실제 인스턴스

## 2. 생성자 `__init__`

-   객체 생성 시 자동 호출
-   인스턴스 변수 초기화

## 3. 클래스 변수 vs 인스턴스 변수

### 클래스 변수

-   클래스 전체에서 공유
-   예: `serial_number`

### 인스턴스 변수

-   객체마다 독립적으로 존재
-   예: `channel`, `volume`, `on`

## 4. self

-   현재 객체 자신을 가리킴.
-   인스턴스 변수와 메서드 접근 시 사용.

## 5. 접근 지정

-   `self.channel` : 공개 변수
-   `self.__channel` : 비공개 변수(캡슐화)

## 6. 메서드

-   Setter: 값 변경
-   Getter: 값 조회

## 7. `__str__`

-   `print(객체)` 시 사람이 읽기 쉬운 문자열 반환.
-   없으면 메모리 주소 형태가 출력됨.

## 8. 원 클래스 예제

`Circle` - `area()` : 원의 넓이 - `circumference()` : 원의 둘레 -
`math.pi` 사용

## 9. 상속

-   `class ElectricCar(Car)`
-   부모 클래스 기능 재사용
-   `super().__init__()`으로 부모 생성자 호출

## 10. 메서드 오버라이딩

-   자식 클래스에서 `__str__()` 등을 재정의 가능.

## 핵심 요약

-   클래스 = 설계도
-   객체 = 실제 생성물
-   `self` = 현재 객체
-   `__init__` = 생성자
-   `__str__` = 출력 형식
-   클래스 변수 = 공유
-   인스턴스 변수 = 객체별 독립
-   Getter/Setter = 데이터 접근
-   상속 = 코드 재사용
-   오버라이딩 = 부모 메서드 재정의
